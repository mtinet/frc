# FRC Display Controller

FIRST Robotics Competition에서 사용할 수 있는 화면 출력기입니다. 점수 입력, 영상/이미지 표시, 별도 모니터 지원 등의 기능을 제공합니다.

## 주요 기능

- 📊 **점수 입력 및 표시**: 두 팀의 점수를 실시간으로 입력하고 표시
- 🎬 **유튜브 영상 재생**: 유튜브 URL을 입력하면 바로 재생
- 🖼️ **이미지 표시**: 로컬 이미지 파일 업로드 및 표시
- 🖥️ **별도 모니터 지원**: 뷰박스 전용 화면을 별도 모니터에 표시
- 🎨 **모던한 UI**: 깔끔하고 직관적인 사용자 인터페이스

## 설치 방법

### 1. Python 설치 확인
Python 3.8 이상이 설치되어 있어야 합니다.

```bash
python --version
```

### 2. 의존성 설치

```bash
pip install -r requirements.txt
```

## 실행 방법

### 기본 실행

```bash
python app.py
```

서버가 시작되면:
- **컨트롤 박스**: http://localhost:8080
- **뷰박스 (별도 모니터)**: http://localhost:8080/view

### 별도 모니터 사용

1. 노트북에서 컨트롤 박스 페이지 열기: http://localhost:8080
2. 별도 모니터에서 뷰박스 페이지 열기: http://localhost:8080/view
3. 뷰박스에서 F11 키를 누르거나 더블클릭하여 전체화면 모드로 전환

## 사용 방법

### 점수 입력
- 각 팀의 +/- 버튼을 클릭하여 점수 조정
- 직접 입력 필드에 숫자를 입력하여 점수 설정
- 팀 이름도 변경 가능

### 유튜브 영상 재생
1. "미디어 컨트롤" 섹션의 유튜브 URL 입력란에 URL 입력
2. "영상 로드" 버튼 클릭
3. 뷰박스에서 자동으로 영상이 재생됩니다

**지원하는 URL 형식:**
- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`

### 이미지 표시
1. "이미지 업로드"에서 이미지 파일 선택
2. 자동으로 뷰박스에 표시됩니다

## 문제 해결

### 포트가 이미 사용 중인 경우
`app.py` 파일의 마지막 줄에서 포트 번호를 변경하세요:
```python
app.run(debug=True, host='0.0.0.0', port=8080)  # 포트 변경
```

### 유튜브 영상이 재생되지 않는 경우
- 영상 업로더가 임베딩을 허용했는지 확인
- 네트워크 연결 확인
- URL 형식이 올바른지 확인
